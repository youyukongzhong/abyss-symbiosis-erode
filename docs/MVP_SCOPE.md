# MVP 范围

## 来自设计文档的核心目标

第一版验证一句话：玩家能否体验到“前期苟且偷生 -> 捡到关键变异 -> 战力突变 -> 扛住撤离尸潮”的情绪曲线。

## 本次已落地系统

1. 搜：地图内散布基因碎片、战术注射器、器官样本和敌人掉落的血肉凝块。
2. 打：玩家俯视角移动射击，敌人包含冲锋、喷吐和雇佣兵远程火力。
3. 撤：地图边缘有净化电梯，启动后进入 60 秒守点撤离。
4. 进化：经验满后暂停并弹出三选一器官，器官会改变属性、攻击方式或外观。
5. 侵蚀：拾取与时间会增加侵蚀度，高侵蚀提高伤害但压缩视野并召唤幻觉。

## 下一阶段建议

1. 将地图从开放场景升级为“房间 + 走廊”的程序化结构。
2. 把器官数据独立成 JSON，方便未来迁移到 Godot Resource。
3. 加入局外避难所、仓库、永久升级和变异图鉴。
4. 加强部位破坏：从弱点暴击扩展到触手、毒腺、护甲等独立部位。
5. 制作真实音效、粒子素材和变异动画。
